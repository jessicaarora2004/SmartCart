 SmartCart: Customer Segmentation & Spending Analysis

 Overview

SmartCart is a data-driven machine learning project that analyzes customer behavior and segments users into distinct groups based on their purchasing patterns, income, and engagement.

The goal is to help businesses **understand their customers better** and enable **targeted marketing strategies** for improved sales and retention.

---

 Problem Statement

Businesses often treat all customers the same, which leads to:

* Inefficient marketing strategies
* Low conversion rates
* Poor customer retention

This project solves the problem by:

> Identifying **hidden customer segments** using unsupervised machine learning.

---

 Solution

We use **K-Means Clustering** to group customers into clusters based on:

* Spending habits
* Income
* Purchase frequency
* Engagement behavior

Each cluster represents a **specific type of customer**, enabling personalized decision-making.

---

Key Features

* Data Cleaning & Preprocessing
* Feature Engineering (Age, Total Spending, Customer Tenure, etc.)
* Data Scaling for fair comparison
* Optimal cluster selection using Elbow Method
* K-Means Clustering
* Cluster Visualization & Analysis
* Customer Behavior Interpretation
